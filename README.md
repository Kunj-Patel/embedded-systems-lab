# Embedded Systems Lab

## Purpose

This repository documents my development as an embedded software engineer through implementation, experimentation, and technical documentation.

The focus is on strengthening my ability to design complete embedded systems by understanding the full stack:

`Hardware → Drivers → Firmware → Real-Time Systems → Applications`

## Areas of Focus

- Modern C++ programming
- Embedded software architecture
- Microcontrollers and hardware interfaces
- Real-time systems and RTOS concepts
- Firmware development practices
- Debugging, testing, and performance analysis

This repository will evolve alongside the projects, experiments, and design notes created throughout this engineering journey.
